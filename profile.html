Profile and data controls

This noindex app-support route shows the profile controls a participant should expect before using local-first demos: export, deletion, visibility, and review paths. The data below is labeled example data.

Account status

Example data Local demo No public profile

Controls

Export local data

Download WRE sessions, dialogue notes, and commitment records as JSON where a tool stores them.

Delete local data

Clear browser-stored demo data, including drafts, examples, and local-only profile state.

Visibility settings

Keep records private by default, with explicit choices for pseudonymous or public visibility.

Review commitments

Request review or revocation for coercion, ambiguity, changed evidence, impossibility, or privacy risk.

Notification controls

Separate contact notifications from public profile visibility and community writing mentions.

Contact support

Use the contact path for deletion, correction, export, or governance questions.
