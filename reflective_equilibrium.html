Reflective equilibrium at scale

Wide reflective equilibrium is difficult for any one person; structured collective deliberation can make parts of the process more tractable. This page explains the method and shows a labeled example session.

Open WRE Agent Tool Read method limits

Method boundary

WRE can surface tensions among judgments, principles, and background theories. It cannot by itself prove a moral conclusion or remove the need for public reasoning.

Example WRE session

Scope: AI hiring assistant. The example uses labeled data to show the workflow without exposing a blank lab or a seeded app session as if it were a public result.

Example data Judgment

Case judgment

A qualified applicant from a lesser-known school should not be screened out solely because the model uses school prestige as a proxy.

Example data Principle

Candidate principle

Hiring systems should avoid proxies that predict opportunity background more strongly than job-relevant ability when direct assessment is feasible.

Example data Background theory

Background commitment

Explainability and auditability are required when automated systems allocate scarce social opportunities.

Example data Tension

Detected tension

The system also values efficient screening. The repair task is to identify a lower-bias screen that preserves enough operational feasibility.

Repair options

Revise the proxy

Replace school prestige with direct work-sample scoring or audited skill measures.

Narrow the principle

Allow proxies only where direct assessment is infeasible and disparate-impact audits pass.

Revise confidence

Lower confidence if operational constraints are stronger than the initial case description assumed.

Known limits

  • Coherence gains can reflect hidden assumptions rather than truth.
  • Input quality matters: vague claims produce vague conflict reports.
  • WRE cannot replace empirical auditing where empirical claims are central.
  • Exports should be reviewed before sharing because belief graphs may contain sensitive commitments.
