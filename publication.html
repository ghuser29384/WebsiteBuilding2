Prestige proxies in first-round screening

Claim: If a hiring system uses prestige proxies in first-round screening, it risks unjust discrimination.

Strongest opposing concern: Some employers use proxies because direct skill assessment is costly and because credentials can carry weak information about training.

Reply: The concern supports better assessment design, not unexamined prestige filtering. A fairer system can test relevant skills directly and audit disparate impact.

Outcome: Narrowed disagreement; neither participant created a follow-through plan.

Publication rules

Label examples

Protect participants
